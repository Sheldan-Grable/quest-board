# Quest Board

Quest Board is a front-end tabletop RPG project for displaying, searching, and filtering fantasy quest cards.

The app is designed like a simple adventurer’s notice board where a Dungeon Master or player can browse available quests by difficulty, quest type, or search terms.

## Live Demo

[View the live project](https://sheldan-grable.github.io/quest-board/)

## Features

* Display a collection of fantasy quest cards
* Filter quests by difficulty
* Filter quests by quest type
* Combine multiple filters
* Search quests by title or description
* Show active filter button states
* Display a quest counter
* Show a no-results message when no quests match
* Use fantasy-inspired parchment styling
